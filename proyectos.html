Web para PetShop

Como proyecto de final de la Segunda Evaluación de Lenguaje de Marcas y Sistemas de Gestión en la UEM, un grupo formado por 4 personas hicimos una página web simulando un pedido de un cliente, en este caso de una clínica veterinaria.
En este proyecto se puso en práctica todos nuestros conocimientos sobre HTML y CSS, así como de CSS Responsive.

Bot Discord para Exile Gaming

En este proyecto, hecho para un equipo de eSports en 2022, se me encargó que bot para la plataforma de Discord, para el servidor de la comunidad del equipo en cuestión. Éste consistía en realizar un matchmaking para que los usuarios jugasen entre sí, creando una cola de búsqueda y posteriormente la sala.
Se trabajó con python, con la biblioteca de Discord.py y con SQL y el servidor remoto de myPhpAdmin.

Proyecto Integrador UEM

Como proyecto de final de primer curso de Desarrollo de Aplicacion Web en la Universidad Europea de Madrid en 2023, un grupo de 4 personas llamados "Daweishine" nos encargamos de crear un programa para administrar los proyectos integradores de toda la universidad. Debía poderse añadir proyectos, eliminarlos y modificarlos, así como los alumnos que participaban en ellos.
Se puso en práctica todo lo aprendido de Java en todo el curso, así como SQL para las consultas con la base de datos. Además, se usó una interfaz gráfica con los componentes de Swing.
